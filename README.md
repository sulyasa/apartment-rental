# 🏠 Apartment Rental - Система аренды квартир на Laravel

Полноценное веб-приложение для аренды квартир с системой агентов, бронированием и личными кабинетами.
https://apartment-rental-production.up.railway.app/ домен

![Laravel](https://img.shields.io/badge/Laravel-10.x-orange)
![PHP](https://img.shields.io/badge/PHP-8.1+-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## 🚀 Возможности

### Для арендаторов
- 🔍 Поиск квартир по городу, типу, цене, количеству комнат
- 📅 Бронирование квартир с выбором дат
- 👤 Личный кабинет с историей бронирований
- 💳 Онлайн-бронирование без комиссии

### Для собственников
- 🏢 Добавление и управление квартирами
- 📊 Просмотр статистики и бронирований
- ✅ Подтверждение/отклонение заявок

### Для агентов
- 📊 Панель управления со статистикой
- 🏠 Управление квартирами клиентов
- 📅 Модерация бронирований
- 💰 Отслеживание доходов и комиссий
- ⭐ Рейтинг и отзывы

## 🛠 Установка

### Требования
- PHP 8.1+
- Composer
- SQLite/MySQL/PostgreSQL

### Шаг 1: Установка зависимостей
```bash
cd apartment-rental
composer install
```

### Шаг 2: Настройка окружения
```bash
copy .env.example .env
php artisan key:generate
```

### Шаг 3: Настройка базы данных

**Для SQLite:**
```bash
touch database/database.sqlite
```

В `.env`:
```env
DB_CONNECTION=sqlite
DB_DATABASE=/полный/путь/к/database/database.sqlite
```

**Для MySQL:**
```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=apartment_rental
DB_USERNAME=root
DB_PASSWORD=
```

### Шаг 4: Запуск
```bash
php artisan migrate
php artisan db:seed
php artisan serve
```

Откройте `http://localhost:8000`

## 👔 Система агентов

### Как стать агентом
1. Зарегистрируйтесь на сайте
2. Войдите в аккаунт
3. Нажмите "Стать агентом" в меню
4. Заполните форму (опыт, лицензия, о себе)
5. Нажмите "Стать агентом"

### Возможности агента
- 📊 Дашборд со статистикой
- 🏢 Управление квартирами
- 📅 Модерация бронирований
- 💰 Отслеживание доходов

### Как повысить эффективность
1. Заполните профиль полностью
2. Добавьте больше квартир
3. Быстро реагируйте на заявки
4. Следите за рейтингом
5. Анализируйте цены рынка

## 📝 API Routes

```
GET    /                      # Главная
GET    /apartments           # Список квартир
GET    /apartments/{id}      # Детали квартиры
POST   /apartments/{id}/book # Бронирование
GET    /my-bookings          # Мои бронирования
GET    /agents               # Список агентов
GET    /agent/dashboard      # Панель агента
POST   /become-agent        # Стать агентом
GET    /login                # Вход
GET    /register             # Регистрация
```

## Система агентов

### Как стать агентом

1. Зарегистрируйтесь на сайте
2. Перейдите по ссылке "Стать агентом"
3. Заполните форму (опыт, номер лицензии, о себе)
4. После одобрения получите доступ к панели агента

### Возможности агента

- 📊 **Дашборд** - Общая статистика
- 🏢 **Квартиры** - Управление своими квартирами
- 📅 **Бронирования** - Подтверждение/отклонение брони
- 💰 **Доходы** - Отслеживание заработка
- ⭐ **Рейтинг** - Рейтинг на основе отзывов

### Как повысить эффективность агента

1. **Заполните профиль** - Добавьте фото, опыт работы, номер лицензии
2. **Добавьте больше квартир** - Больше квартир = больше бронирований
3. **Быстро реагируйте** - Подтверждайте бронирования в течение дня
4. **Следите за рейтингом** - Высокий рейтинг привлекает больше клиентов
5. **Обновляйте цены** - Анализируйте рынок и корректируйте цены
