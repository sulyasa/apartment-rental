# 🚀 Инструкция по загрузке на Beget

## Шаг 1: Регистрация и создание сайта

1. Перейдите на https://beget.com/
2. Зарегистрируйтесь (или войдите)
3. В панели управления выберите **"Создать сайт"**
4. Выберите **"Подключить домен"** или используйте тестовый домен
5. Выберите **"PHP"** как тип сайта

## Шаг 2: Подключение GitHub (рекомендуется)

### Способ А: Через панель Beget

1. В панели Beget перейдите в **"Сайты"**
2. Нажмите на ваш сайт
3. Выберите **"Git"** в меню
4. Введите URL репозитория:
   ```
   https://github.com/sulyasa/apartment-rental.git
   ```
5. Укажите ветку: `master`
6. Нажмите **"Подключить"**

### Способ Б: Ручная загрузка через FTP

1. В панели Beget перейдите **"FTP"** → **"Настройки"**
2. Получите данные для подключения:
   - Хост: `ftp.ваш-домен.beget.ru` или `ваш-домен.beget.ru`
   - Логин: ваш логин на Beget
   - Пароль: ваш пароль

3. Используйте FileZilla:
   - Скачайте: https://filezilla-project.org/
   - Подключитесь к серверу
   - Загрузите все файлы из папки `apartment-rental/` в папку `public_html/`

## Шаг 3: Настройка базы данных

1. В панели Beget перейдите **"Базы данных"**
2. Создайте новую базу данных MySQL
3. Запомните:
   - Имя базы данных
   - Имя пользователя
   - Пароль

## Шаг 4: Настройка .env

1. В FileZilla отредактируйте файл `.env` (или создайте его):
```env
APP_NAME=ApartmentRental
APP_KEY=base64:генерируйте_свой_ключ
APP_DEBUG=true
APP_URL=https://ваш-домен.ru

DB_CONNECTION=mysql
DB_HOST=localhost
DB_PORT=3306
DB_DATABASE=имя_бд
DB_USERNAME=пользователь_бд
DB_PASSWORD=пароль_бд
```

## Шаг 5: Запуск миграций

1. В панели Beget перейдите **"SSH"** (если доступно)
2. Выполните команды:
```bash
php artisan migrate
php artisan db:seed
```

Или откройте в браузере:
```
https://ваш-домен.ru/migrate.php
```

## Шаг 6: Проверка

1. Откройте ваш домен
2. Сайт должен работать!

---

## Альтернатива: если нет SSH

Создайте файл `migrate.php` в корне сайта (уже есть в проекте) и откройте его в браузере.
